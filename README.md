# 🎵 练琴计时器 - 个性化练习计划 & 专注度检测

这是一个纯网页项目，打开浏览器就能使用，无需安装任何软件。

## 功能概览

- **自定义练习阶段** — 添加练习/休息阶段，设置名称和时长
- **倒计时器** — 开始/暂停/继续/重置，支持自动和手动切换模式
- **任意时候切换下一阶段** — 练习过程中可随时点击"下一阶段"提前切换
- **语音播报** — 阶段切换时自动播报名称
- **钢琴频谱专注度检测** — 基于音量阈值、钢琴频段能量和频谱峰值的检测，可减少环境噪音误判
- **环境噪音校准** — 一键校准当前环境噪音水平
- **节拍器** — 支持 BPM 调节、拍号选择、强拍弱拍区分，可与练习阶段联动
- **练习完成总结** — 显示总时长、完成阶段数、提前切换次数、专注提醒次数等

## 给体验用户的使用说明

### 如何开始使用

1. **打开网页链接** — 在浏览器（推荐 Chrome 或 Edge）中打开项目链接
2. **添加练习阶段** — 在"添加练习阶段"区域，输入阶段名称（如"音阶练习"）、设置分钟数，点击"添加阶段"
3. **开始练习** — 添加好阶段后，点击"开始练习"按钮，计时器就会开始倒计时
4. **切换模式** — 可以选择"自动切换"（一个阶段结束自动进入下一个）或"手动切换"（阶段结束后自己选择下一步操作）

### 关于专注度检测

- 点击"开启麦克风检测"可以使用专注度检测功能
- 浏览器会弹出询问"是否允许使用麦克风"，请点击"允许"
- 如果点击了"不允许"或"拒绝"，专注度检测无法使用，但计时器仍然可以正常使用
- 建议在安静环境下使用，检测结果会更稳定
- 在钢琴频谱检测模式下，建议先点击"环境噪音校准"按钮进行校准

### 关于节拍器

- 设置 BPM 和拍号后，点击"开启节拍器"即可使用
- 勾选"练习阶段自动开启节拍器"，则练习阶段自动开启、休息阶段自动关闭
- 暂停练习时节拍器也会暂停

### 如果麦克风不可用怎么办

- 专注度检测只是一个辅助功能，不是必须的
- 即使没有麦克风，你仍然可以正常使用计时器来规划练习时间
- 如果使用的是 HTTP 网页（不是 HTTPS），麦克风功能可能无法使用，请使用正式发布的 HTTPS 链接

## 部署到 GitHub Pages

本项目是纯 HTML/CSS/JavaScript 项目，适合部署到 GitHub Pages。步骤如下：

1. **创建 GitHub 仓库** — 在 GitHub 上创建一个新的仓库
2. **上传文件** — 将以下文件上传到仓库：
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. **打开仓库 Settings** — 进入仓库的 Settings 页面
4. **进入 Pages** — 在左侧菜单找到 Pages
5. **选择部署源** — Source 选择 main branch，文件夹选择 root
6. **保存** — 点击 Save，等待几分钟生成网页链接

部署完成后，你会得到一个类似 `https://你的用户名.github.io/仓库名/` 的链接，分享这个链接给任何人就能直接使用。

## 常见问题记录

### 1. API Key 不要泄露
API Key 是访问 DeepSeek 等服务的凭证，切勿将其硬编码在代码中或上传到公开仓库。建议使用环境变量或配置文件管理，并确保 `.gitignore` 已忽略相关文件。

### 2. Git 需要安装才能使用 Checkpoints
Cline 的 Checkpoints 功能依赖 Git 来管理代码快照。如果未安装 Git 或未在项目中初始化 Git 仓库，Checkpoints 将无法正常使用。请确保已安装 Git 并在项目根目录执行 `git init`。
